## `> ls ./projects`

### 🔗 IntegracionEcommerceFenix
> **C# Console App** — Integración automática de órdenes de e-commerce con el ERP.

Procesa pedidos entrantes, transforma y valida los datos, e inserta registros en tablas de encabezado, detalle e impuestos del ERP. Manejo de campos personalizados como cantidades de embalaje y asociación de empleados por orden.

`C#` `.NET Core` `SQL Server` `ERP Integration` `Clean Code`

---

### 📥 Smart Excel Importer
> **Web App** — Carga drag-and-drop de archivos Excel con previsualización de datos e inserción a base de datos.

Interfaz web que permite importar archivos `.xlsx` con validación previa, previsualización en tabla interactiva y pipeline de inserción hacia SQL Server.

`ASP.NET Core` `JavaScript` `SQL Server` `File Processing`

---

### ⏱ Time Tracking System
> **ASP.NET MVC App** — Sistema interno de registro y supervisión de horas laborales.

Incluye autenticación por roles (empleados / supervisores), rutas protegidas por middleware, agrupación de registros por departamento y visualización tipo acordeón por código de empleado.

`ASP.NET MVC` `SQL Server` `Role-based Auth` `Middleware`

---

### 🔄 Employee Sync Program
> **C# Console App** — Sincronización bidireccional de empleados entre sistemas.

Sincroniza registros entre `TimeLog` y `BioTime` detectando altas, bajas y modificaciones. Muestra resumen de cambios y se cierra automáticamente tras completar el proceso.

`C#` `.NET` `SQL Server` `MySQL` `Data Sync`
